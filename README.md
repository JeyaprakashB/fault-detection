# fault-detection
# ⚡ Smart Grid Fault Detection System

## 📌 Description
This project simulates a Smart Grid Fault Detection System using software.  
It monitors electrical parameters like voltage and current to detect faults in the system.

---

## ⚡ Features
- Real-time Voltage & Current input
- Fault Detection:
  - Overvoltage
  - Undervoltage
  - Overcurrent
  - Short Circuit
- Load Analysis (Low / Normal / High)
- User-friendly GUI (CustomTkinter)
- Auto input simulation
- Status display and alerts

---

## 🧠 Working Principle
The system compares input voltage and current values with predefined thresholds.  
Based on conditions, it identifies the type of fault and displays appropriate alerts.

---

## 🛠 Technologies Used
- Python
- CustomTkinter (GUI)
- Basic Electrical Concepts
